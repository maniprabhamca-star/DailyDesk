PRO

Pro differentiators — the moat rivals can't copy

Genuinely unique to DiemDesk's on-device architecture — planned as the flagship Pro hooks. Server-first competitors (Smallpdf/iLovePDF/Adobe) would have to rebuild their whole stack to match these.

On-device batch
Process 100 files at once — zero uploads. Their batch = 100 files on their servers; they physically can't match it.
Redaction certificate
Signed, exportable proof a doc was truly scrubbed (0 recoverable chars, metadata stripped) — a legal/compliance wedge.
Saved workflows
One-click document assembly line — merge · clean · sign · share-safe · compress — run on a whole batch.
Encrypted File Vault
Zero-knowledge E2E storage (Argon2 + AES-256) — even we can't read it. Rivals' storage is server-readable.
Self-destruct shares
Expiring, password-locked share links that DiemDesk itself can't decrypt.
Certificate of completion
DocuSign-style signed audit trail + verification hash for e-signatures.
Bates numbering
Legal page-stamping — a standard paid feature that pulls in the legal vertical.
Brand kit
Save logo / header-footer / watermark once, apply across a whole batch.
Offline PWA
Install it; works fully offline. Server-first rivals go dark without a connection.
True re-encode Edit
Higher-fidelity in-place text-editing tier for Pro.

Tool roadmap — Phase 1 (building now, one at a time)

Research-grounded from real global file pains (see docs/designs/tool-expansion-roadmap.md). #1–5 ship FREE & public as each is finished (owner gets a screenshot + test URL each time); #6 PDF→Excel is the Pro hook, gated until the Pro launch.

1 · Compress to target size FREE · mockup ✓
Hit an exact KB/MB limit for exam/visa/court portals (UPSC 40 KB, SSC 100 KB…). PDF & image. #1 global pain + huge SEO.
2 · Passport / ID photo FREE
Per-country specs + background + size-to-KB. Faces stay on-device — our privacy edge.
3 · Fill PDF form FREE
Type / checkbox / date / sign on any PDF, then flatten. Closes a headline free-tier gap.
4 · EXIF strip + image blur FREE
Strip GPS/metadata + blur faces/plates. On-brand privacy chores nobody else leads with.
5 · Dev + CSV pack FREE
base64, hash, JWT, timestamp, UUID, diff, YAML↔JSON, CSV↔Excel, minifiers — cheap keyword magnets.
6 · PDF → Excel PRO · gated
Tables / bank statements → spreadsheet. The Pro revenue hook; reveal at the Pro launch.
