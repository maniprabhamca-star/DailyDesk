What's new in v1.1 (vs. the live tool)

  • 1
    Read the real fields. We now read each AcroForm widget's box, type and page — so text, checkbox, dropdown and radio fields all light up exactly where they sit.
  • 2
    Click & Tab like a native form. Click a field to type into it; Tab/Shift+Tab walk through fields in reading order. No more eyeballing where to drop text.
  • 3
    Keep-editable export. Values are written back into the AcroForm, so the download stays a fillable form — the recipient can still change or sign it. This is the headline v1.1 win.
  • 4
    Flatten is now a choice. The old always-flatten behaviour becomes the "Flatten & lock" option for when you want the answers final. Flat scans keep the place-anywhere tools unchanged.
