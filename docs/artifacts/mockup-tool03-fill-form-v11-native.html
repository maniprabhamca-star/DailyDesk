What's new in v1.1

Everything the live tool does today stays — this adds real-field intelligence and full control on top.

  • 1
    Reads the real fields. Each AcroForm widget's box, type, page and required-flag — so text, checkbox, radio, dropdown and list fields all light up exactly where they sit.
  • 2
    Click & Tab like a native form. Click to type; Tab/Shift+Tab walk fields in reading order; a field list shows what's filled, empty or required.
  • 3
    Keep-editable export. Values are written back into the AcroForm, so the download stays a fillable form — the recipient can still change or sign it. The headline v1.1 win.
  • 4
    Flatten is now a choice. The old always-flatten becomes “Flatten & lock” for when you want answers final. Flat scans keep today's place-anywhere tools untouched.
  • Per-field control — value, font, size (auto-fit or fixed), colour, alignment, clear & reset.
  • Every field type — checkbox mark style, radio option pick, dropdown select (with optional custom typed value), multi-select lists.
  • Apply-to-all defaults, highlight on/off, Enter-to-advance, undo, and a stable filename.
  • Signature-lock and appearance-rebuild options so the file looks right in every PDF viewer.
