# DiemDesk artifacts

All the visual HTML artifacts (overviews + tool mockups) in one place. Open any file
in a browser, or use its live link. **Naming convention:**
- `overview-*.html` — product/status overviews.
- `mockup-toolNN-<slug>.html` — a tool's approved design mockup (NN = build order).

Keep the live links stable by re-publishing to the **same** URL (the artifact tool
takes the existing URL) whenever a file changes.
