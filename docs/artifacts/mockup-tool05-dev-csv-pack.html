Why this pack, and how it stays clean

  • 1
    One template, fifteen tools. Identical header, Input→Output panels, copy button and spacing everywhere — so the pack reads as a single polished product, never a cluttered grab-bag.
  • 2
    Options only when needed. Base64 shows an Encode/Decode switch; UUID shows a count; most tools show nothing but the two panels. No knobs for the sake of knobs.
  • 3
    Each is its own keyword URL (/base64, /csv-to-json, /uuid-generator…) — the whole point: cheap, client-side pages that each rank for a specific developer search.
  • 4
    100% on-device & free. Nothing is uploaded; reinforces the privacy hook. A small hub page links them all, grouped as above.
