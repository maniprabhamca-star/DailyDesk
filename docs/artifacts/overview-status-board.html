Pro roadmap — tiered & revenue-safe

Deterministic features cost $0 to serve (pure margin) — the priority; server/AI features are metered so cost never exceeds revenue. Status: BUILT ✓ = working, owner-gated until launch · PARTIAL = partly shipped · PLANNED = not built. (Bates numbering shipped FREE — it's no longer a Pro item.)

🟢 Tier 1 — Deterministic (zero cost · build next)

Self-destruct encrypted shares PLANNED
Best new idea. Expiring, password-locked links DiemDesk can't decrypt — builds on Vault crypto; every recipient sees DiemDesk (viral loop).
Saved workflows + presets PLANNED
Chain tools (compress → Bates → watermark) + save configs (a Bates scheme, redaction set, compress profile). Pure retention; uses the existing file-handoff plumbing.
Trust pack — redaction certificate + RFC-3161 timestamp PLANNED
Signed proof a doc was truly scrubbed + cryptographic proof it existed at a time. Legal/finance pay real money for provable actions.
On-device batch PARTIAL
100 files at once, zero uploads — live on 6 tools already; extend to all. Rivals' batch uploads everything; they can't match it.
Certificate of completion (e-sign) PLANNED
DocuSign-style signed audit trail + verification hash for signatures.
Brand kit PLANNED
Save logo / header-footer / watermark once, apply across a whole batch and Link-in-Bio.

🟣 Tier 2 — Server / AI (metered + capped)

AI document suite (8 tools) BUILT ✓
Chat / Summarize / Translate / Q-gen / AI-redact / compare / Excel-cleanup / ⌘K. Haiku + per-user cap + $5/day global kill-switch. Gated until launch.
Encrypted File Vault BUILT ✓
Zero-knowledge E2E storage (Argon2 + AES-256) — even we can't read it. Phases 1–4 shipped dark.
Bank Statement Converter PARTIAL
The paid flagship (~70% built) → Excel/CSV/Tally, India-first. Own metered tier (₹1,499/$19).
Encrypted "send large file" PLANNED
Privacy-first WeTransfer on Vault storage — quota-capped. Another viral surface.
PDF → Audio MP3 export PLANNED
Downloadable MP3 via server neural TTS (free read-aloud is already live at /pdf-to-audio).
e-Invoice (Factur-X / ZUGFeRD + India GST) PLANNED
Regulatory mandates 2026–28 = non-discretionary B2B spend. Save until the flagship proves India B2B appetite.

FLAGSHIP

Bank Statement Converter — the Pro conversion engine

The paid flagship (spec 2026-07-16 — docs/designs/bank-statement-converter.md). The 57 free tools are the funnel; this is what people pay for. Statement PDF → balance-verified transaction table → Excel / CSV / Tally XML / QBO / OFX, India-first (SBI, HDFC, ICICI, Axis, Kotak…).

The wedge
The only converter that (a) processes locally, (b) validates every row against the running balance, and (c) exports to Tally — which every Indian CA uses and no competitor supports.
Why it earns
The searcher is a business with a deadline. Rivals prove the price: DocuClipper $29–159/mo, CapyParse $29, BankStatementLab $9. Ours: free 5 pages · ₹399/$4.99 pack · Statements Pro ₹1,499/$19/mo (NOT bundled into $5.98 Pro).
Privacy finally load-bearing
A bank statement is the most sensitive doc anyone uploads — and every rival uploads it. "Never leaves your browser" stops being a nice-to-have and becomes the reason to buy.
~70% of the engine exists ✓
Shipped this week for PDF→Excel: positional extraction, phantom-column suppression, form-vs-table detection, XLSX/CSV, editable grid — plus unlock-pdf, OCR, AI + cost caps, Stripe. New: bank fingerprinting · column maps · balance engine · Tally XML · bank SEO pages.
Balance check = free accuracy oracle
It's not just a trust badge — it auto-verifies extraction. So: local → Haiku → escalate to Sonnet only when the balance fails. Keeps AI cost near zero instead of eating the $19.
Critical path
Real statement samples per bank (not code) + validating Tally XML against a real Tally Prime import. ⚠ Sequencing: this is the revenue engine; native apps only distribute a product that doesn't monetise yet.
